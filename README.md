## Overview
This project presents an Exploratory Data Analysis (EDA) of one year's worth of sales data from a fictitious pizza restaurant. The analysis was carried out using Python in a Jupyter Notebook to answer key business questions and uncover meaningful insights from the sales data.
## Objectives
The analysis covers the following:
- Calculate the total revenue generated.
- Determine the total quantity of pizzas sold.
- Determine the total number of customer orders.
- Identify the number of pizza types sold.
- Calculate the average price of pizzas.
- Identify the peak hours of sales.
- Analyze total sales by day of the week.
- Identify the top five bestselling pizzas.
- Analyze monthly sales trends.
- Identify underperforming pizzas.
- Perform additional exploratory analyses and visualizations.
## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
## Dataset
The project uses the **Pizza Place Sales** dataset, which consists of four related CSV files:
- Orders
- Order Details
- Pizzas
- Pizza Types
These datasets were merged into a single DataFrame for analysis.
## Key Insights
The analysis provides insights into:
- Revenue performance
- Customer ordering patterns
- Peak business hours
- Weekly and monthly sales trends
- Bestselling and underperforming pizzas
- Sales by pizza category and size
